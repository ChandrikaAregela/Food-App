# Food Ordering App

## About the Project

This is a simple console-based Food Ordering App developed using Python. The application allows users to register, log in, browse available food items, add items to a cart, and place orders. All data is stored using JSON files, making the project lightweight and easy to understand.

I created this project to improve my understanding of Python programming, file handling, modular programming, and basic Object-Oriented Programming concepts.

---

## Features

* User Registration
* User Login
* View Available Food Items
* Add Food Items to Cart
* View Cart
* Place Orders
* View Order History
* JSON-based Data Storage

## How It Works

1. A user registers with a username and password.
2. The user logs into the application.
3. Available food items are displayed from the food database.
4. The user can add food items to the cart.
5. The cart displays selected items and the total amount.
6. The user places an order.
7. Order details are saved in the order database.
8. Users can view their previous orders.

---

## Data Storage

The project uses JSON files to store data:

* `users.json` – Stores user credentials.
* `foods.json` – Stores food menu information.
* `orders.json` – Stores order history.

No external database is required.

## How to Run the Project

1. Download or clone the project.
2. Open the project folder in VS Code.
3. Open the terminal.
4. Run the following command:

```bash
python main.py
```

5. Follow the menu options displayed on the screen.
